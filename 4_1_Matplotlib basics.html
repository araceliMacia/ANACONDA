Matplotlib basics¶

Matplotlib has become the de facto standard for scientific 2D plotting in Python. It originally attempted to provide a style of interaction similar to Matlab. It allows you to save graphics to different raster and vector image formats, including PDF, JPG, SVG, PNG, BMP and GIF.

Plots and visualizations are one of the most important tasks in data analysis. Having an understanding of the basics of matplotlib is thus an essential aspect of data science in Python.

En la pagina matplotlib hay una seccion llamada Galery, que ya estan presideñados, y se coge el codigo para personalizarlo.¶

Plotting and attributes...¶

Using matplotlib requires typically importing the pyplot object. There is also the option of starting IPython in "pylab" mode, but we will not cover it here. You can plot in a single cell by simply using that object. There are different ways of changing the properties of lines...

import numpy as np

# Get some random points from a gamma dist.

# Tell IPython to use the internal GUI (there are other options, but require correct install of external software)
%matplotlib inline

# Plot and change the line using a format string:
#line, = plt.plot(L, "--")  # Note the comma!

# This is the same as:  line, la coma es para que devuelva la primera linea.

plt.setp(line, color="r")

# Method 3: Call directly the set_XX on the object:   y la propia linea tiene propiedades.

line.set_linewidth(2.0)

# Change figure-related properties:
plt.title("My gamma sample")
plt.ylabel("some values")

What if you want to do something else with the plot above?¶

You can change plot properties overall and of the different parts of the plot if you are in the same Notebook cell as many times you want. This means the state of the plot is maintained in the same cell.

But by default, IPython closes the previous plot, so it is no longer accessible, a new one is created. This is to avoid interference between cells.

**la norma : las configuraciones solo afectan a una "celda" del notebook.. una celda genera un grafico, y en otra celda el grafico no existe.

In [23]:
plt.title("title changed")
plt.show()

However, you can enable the configuration of IPython so that it does not close previous plots. This can be achieved with a configuration option. **con lo siguiente, se cambia el comportamiento del notebook para que en celdas distintas se pueda toquetear un grafico. No es recomendable!!.

In [24]:
from IPython.core.display import set_matplotlib_close

set_matplotlib_close(False)
# Now we plot again.

# Clear the figures before:
plt.clf()
plt.plot(L)
plt.show()

And now we change the plot, and it changes the previous one.

In [25]:
plt.title("We changed the title")
plt.plot([i+0.5 for i in L])
# There can be several lines in the same plot:
plt.show()

Now let's revert to the normal behaviour.

In [26]:
set_matplotlib_close(True)
plt.clf()

Working with subplots¶

Using the subplot function, you can have several subplots in the same figure. Una sola figura con varios graficos dentro.

In [30]:
plt.figure(1)  # Creates a figure, this is done by default, you can omit it.
plt.subplot(224) # 2 rows, 2 columns, select the last one. >> 224 : dos filas, dos columnas y posicionate en la 4. 

# Create some data and plot it:
data = np.random.random(100)
x = range(len(data))
plt.plot(x, data) # este lo pinta en el cuarto cuadrado.

# Now select other subplot:
plt.subplot(221)
plt.scatter(x, data)  #este plot lo pinta en la 2 filas, dos columnas y lo pinta en el primer cuadrante.

Inside Matplotlib objects¶

Instead of having a single plot, it is often convenient to have several and manipulate them explicitly. You can create several plots using the figure function.

ax.set_title("Again my gamma")

# You can change also Figure properties:
fig.set_facecolor("y")
