Merge es el Join entre dos tablas. Con how le decimos el comportamiento. Outer, quiere decir, que la tabla resultante meta todos los datos, con que este en una de ellas basta, y tambien se puede poner left o right, que incluya los que esten una tabla o la otra. una funcion ttest_ind para evaluar dos arrays ( trabaja con Numpay). bydisease.groups.values() esto da las series de datos y se meten en la funcion.

Let's check if the mean of both subpopulations is the same.

In [17]:
from scipy.stats import ttest_ind
g1, g2 = bydisease.groups.values()
ts, p = ttest_ind(g1, g2)
print p
print 'Difference!' if p < 0.05 else 'No difference'

0.00200590028439
Difference!

From the doc of ttest_ind: If the p-value is smaller than the threshold, e.g. 1%, 5% or 10%, then we reject the null hypothesis of equal averages.
